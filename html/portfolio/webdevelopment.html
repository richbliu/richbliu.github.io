Biomed Resource Website

www.bmres.com Biomed Resource needed to redesign their existing site to reflect the expanded disposables product line and company profile information in a clean and efficient manner. A green color scheme was chosen, and a jquery slideshow was placed on the frontpage to showcase their newest products.

Weiss Tech House Website

www.tech-house.upenn.edu As a student leader of the Weiss Tech House, I chose to redesign the outdated website and create a modern, highly-functional site that followed George Weiss's vision: to create an innovation hub for all students. The Weiss Tech color scheme runs uniformly throughout the site, and the layout is centered around upcoming events and competitions.

GridNavigator Website

www.gridnavigator.com GridNavigator, an energy forecasting and management company, requested a modern, clean website which reflected their cutting-edge technology. The navigation is designed to be intuitive, and the site was designed using responsive HTML5 to ensure proper display on all mobile devices.

MyIvyEdge Website

www.myivyedge.com An entire website redesign was executed for MyIvyEdge, an ivy-league preparation consulting company. The user interface was formed around the existing logo, and jquery elements were implemented into mentor profiles and forms to allow for simplified sorting and navigation.

Invisergy Corp Website

www.invisergycorp.com Invisergy, a clean energy company with innovative solar window technology, looked for a trendy, modern site that would highlight their main product while providing a landing page for investors and customers. To allow for easy maintenance, the website was created using Wordpress' CMS framework.

Graphene Frontiers Website

www.graphenefrontiers.com Graphene Frontiers, a materials manufacturing corporation, asked for a website that would reflect their product, graphene, while looking professional for investors and customers. The website was conceived to be updated in virtually any operating environment (mobile, tablet, desktop) so the most updated news could get fed to the website instantly.

CytoVas Website

www.cytovas.com CytoVas, an invitro diagnostics company, provides high technology solutions for investigating cardiovascular health assays. The company asked for a "simple, effective" website for highlighting their service and attracting investors. The website is coded in HTML5 and CSS.

PennEWB Website

www.pennewb.org Penn Engineering without borders needed a complete site redesign that would allow for easy blogging updates. The site was migrated from a HTML/CSS framework to a Wordpress CMS with custom plugins to simplify web maintenance.

PennLaunch Website

www.pennlaunch.com PennLaunch is a resource to connect entrepreneurs with entrepreneurs within the Penn community. The website was inspired by social networks, such as Facebook and LinkedIn, and includes an advanced team formation feature with substantial user profile options. A large marketing campaign followed, and over 1000 new users signed up for the site within a year.

Looche Website

www.looche.com Looche is a social network built for corporations and consumers invested in saving energy. The goal was to create a customized front-end so that users could easily check their race calendar, race locations, and current standings. Once logged in, users can monitor their energy usage and compete with others for energy credits.

Skytex Website

www.skytex.com Skytex is a tablet manufacturer based in Hong Kong. The company asked for a website inspired by Samsung, driven by Apple, yet enthused with Skytex. They have several tablet lines, and the website was designed to showcase each model in a trendy and unique way.

UPennPost Website

www.upennpost.com UPennPost is designed to be a classifieds site exclusively for Penn students (upenn edu required). The objective was to create an aesthetically pleasing site that operated efficiently within the Penn community. Users can create detailed profiles and submit items for sale or wanted using an easy 1-2-3 submission process.

Dellic Website

www.dellic.com Dellic is an online consumer platform for users to submit ideas for retailers and corporations. A secure idea submission process was developed, and the website allows for full monitoring, from idea submission to processing and feedback.

QuantXchange Website

www.quantxchange.com QuantXchange, an independent trading alpha research firm, asked for a simple and intuitive website which showcased their products and services. A logo was designed, and a mobile compatible website was developed and optimized for SEO.

Bayside Graphics Website

Bayside Graphics was formed to deliver high-impact web development solutions for startups and mid-sized companies. A full-featured portfolio website was developed, along with a magento-powered e-commerce solution to simplify online ordering.

WSCO Website

www.whartonsupplychain.org Wharton Supply Chain Organization, WSCO, holds one of the largest annual supply-chain conferences on campus. The board seeked a modern website which would accelerate their recruitment and marketing efforts. A full-featured wordpress site was designed to facilitate with maintenance and ease of use.

AlphaPoint Legacy Solutions

www.alphapointls.com/ AlphaPoint Legacy Solutions provides solutions for analyzing, renovating, and remediating legacy code. A comprehensive website was designed for promoting their services and showcased the different programming languages which they handled.

PIBV Website

www.pibv.org Penn International Business Volunteers, PIBV, is a student-run organization on campus that specializes in solving the operational problems of NGOs - while enhancing their sustainability and social impact. A highly visible website was developed to promote member interaction and sponsor recruitment.
